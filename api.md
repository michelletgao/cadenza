# API Specification
## Overview of app

***
## **Get all users**
**GET** /api/users/

##### Response
 ```yaml
{
    "success": true,
    "data": [
        {
            "id": <ID>,
            "name": <USER INPUT FOR NAME>,
            "username": <USER INPUT FOR USERNAME>,
            "requests": [ <SERIALIZED REQUESTS>, ... ],
            "fav_songs": [ <SERIALIZED SONGS>, ... ],
            "friends": [ <SERIALIZED USERS WITH JUST NAMES, USERNAMES>, ... ]
        },
        {
            "id": <ID>,
            "name": <USER INPUT FOR NAME>,
            "username": <USER INPUT FOR USERNAME>,
            "requests": [ <SERIALIZED REQUESTS>, ... ],
            "fav_songs": [ <SERIALIZED SONGS>, ... ],
            "friends": [ <SERIALIZED USERS WITH JUST NAMES, USERNAMES>, ... ]
        },
        ...
    ]
}
```

## **Create a user**
**POST** /api/users/

##### Request
 ```yaml
{
    "name": <USER INPUT FOR NAME>,
    "username": <USER INPUT FOR USERNAME>
}
```

##### Response

 ```yaml
{   
    "success": true,
    "data": {
        "id": <ID>,
        "name": <USER INPUT FOR NAME>,
        "username": <USER INPUT FOR USERNAME>
    }
}
```
